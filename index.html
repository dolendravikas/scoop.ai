About Scoop

What is Scoop?

Scoop is an intelligent social media aggregation platform that helps you gather and analyze information from multiple social media platforms using AI-powered analysis.

Features

  • Search across X (Twitter), LinkedIn, Reddit, and Quora
  • AI-powered analysis with multiple models
  • Time-based filtering and keyword focus
  • Voice input and dictation support
  • Real-time intelligence reports

How to Use

  1. Enter your query in the search box
  2. Select platforms using the globe icon
  3. Choose your preferred AI model
  4. Set time range and keywords if needed
  5. Click search or use voice input
